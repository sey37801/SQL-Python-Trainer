#Python und SQL Trainer - Nutzbar aber noch nicht ganz fertig - trotzdem sehr Hilfreich für Anfänger
Python: Für absolute Anfänger: Drücke 3 x auf Hilfe und du bekommst die Lösung, jedoch keine Punkte
Tägliche Wiederholung führt zu Einprägung, eingeprägt kann beim nächsten Durchgang weniger Hilfe 
angefordert werden und die Gesamtpunkte steigen, von Woche zu Woche. 
Der Gedanke ist: Wenn alles verstsanden wurde kannst du kleine Anwendungen sehr viel besser verstehen und siehst es an deinen Punkten.
Wichtig: Lerne anfangs außerhalb der Anwendung wie das Python-gerüst aussehen soll: 
                                                                                      1. Was soll das Programm tun?
                                                                                          2. Welche Hauptschritte gibt es?
                                                                                              3. Welche Schritte verdienen eine eigene Funktion?
                                                                                                 4. Funktionen implementieren.
                                                                                                     5. Alles in main() zusammensetzen.
                                                                                                         6. Testen.

Vorsicht: richtiges Einrücken ist erforderlich um die Aufgabe zu lösen
Reset All: Alles wird zurückgesetzt
Code entspannt und ordne ihn stets übersichtlich an, später kann man ihn immer weiter ausbauen



SQL: Hier lernst du einfache Befehle aber kein vollständiges Erstellen einer Datenbank.
SQL: Du brauchst Grundlagenwissen, deshalb gilt der Trainer noch als unvollständig, in der nächsten Version wird SQL erweitert.
